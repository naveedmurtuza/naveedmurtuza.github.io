  • Two-factor Credential Provider (Vista/7/8)

    Description

    The Two-factor Credential Provider aims to improve the overall security of the windows logon process by adding one time password to windows logon screen. The windows default authentication ("something you know") and the second factor ("something you have", OTP) are independent of each other, but both are required to authenticate the user. The OTP is provided by the otp hardware token.A high precision caliberated real time clock is connected to the server and synchronized with the otp hardware token.A RPC service running on the server verifies the OTP.

    Responsibilities
    • Designed and implemented the windows credential provider using C++
    • Implemented customised time based OTP algorithm
    • Using C++, implemented RPC to communicate with credential provider and verify the otp
